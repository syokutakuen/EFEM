EFEM:線形弾性解析のためのプログラム。名前はEasy-FEM Program（安直なFEMプログラム）から。

作成の動機
　どことは言わないが、余りに下手糞なプログラムが掲載された書籍を見たから。
  この期に及んであの程度とか、人をからかってるんですかね。

使用言語
　CかC++。Fortran?死んでろ。
　データー構造とかそういうのを意識した造りにする。
  また、拡張が容易になるように配慮した。

機能
　二次元弾性解析（平面応力と平面ひずみ・軸対称ソリッド）
  等方性物体のみ使用可能。
　節点荷重と偏荷重・体積力が使用可能。
  通常の拘束変位の他、強制変位を掛けることが可能。
　全体剛性行列は上三角のみ使用したものを解く。ハードコードしているのはエレガントじゃないなー・
